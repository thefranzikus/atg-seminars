# ATG Seminars Website

This is the ATG Seminars training website. It showcases upcoming seminars, training methodology, and allows users to join the waitlist.

## Features
- Hero section with call-to-action
- Upcoming seminars carousel by location
- Training methodology overview
- FAQ section
- Waitlist signup form

## Deployment
This site is deployed on Vercel and automatically updates when changes are pushed to the main branch.
